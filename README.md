# Elite Fullstack Template

A production-grade, immersive fullstack architecture designed for high-performance applications.

## 🚀 Frontend (Next.js)
Located in `/frontend`, this is a React-based immersive environment.

### Features
- **Immersive Styles**: Neumorphism, Glassmorphism, Brutalism, Claymorphism.
- **Animation**: Framer Motion & GSAP for cinematic transitions.
- **3D**: Three.js (React Three Fiber) integration for WebGL scenes.
- **UI Kit**: Tailwind CSS + Lucide Icons.

### Getting Started
```bash
cd frontend
npm install
npm run dev
```

## ⚙️ Backend (FastAPI)
Located in `/backend`, this is a Python-based high-performance API.

### Architecture
- **Clean Architecture**: Decoupled layers (Service, Repository, API).
- **Security**: JWT Authentication with OAuth2.
- **Data Layer**: SQLAlchemy ORM with Pydantic schemas.
- **Real-time**: WebSocket ready.

### Getting Started
```bash
cd backend
.\venv\Scripts\activate
pip install -r requirements.txt
python app/main.py
```
